<h2>✨ Instalação</h2>

Instale as dependencias utilizando:
```bash
yarn
```

<h2>✨ Executando</h2>

Ápos a instalação das dependemcias, execute o projeto utilizando:

```bash
yarn start
```

 <h3>❗O projeto é executado na porta 3000, portanto, verifique se ele está livre para uso.</h3>
